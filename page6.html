Microsoft Azure

The motive of the talk was to create an awareness about the Imagine Cup and learn the fundamentals of Microsoft AZURE. The session on Introduction to Imagine Cup was conducted by Mr. Venkatesh Sarvasiddhi & Ms. Osheen Chavhan. The following session about the fundamentals of Microsoft AZURE was conducted by Mr. Chaitanya Dandvate.
